Converter API

Universal file conversion API supporting 100+ formats including images, documents, audio, and video. High-fidelity processing with minimal latency.

POST /convert
curl -X POST https://api.hqmx.net/converter/convert \ -H "Content-Type: multipart/form-data" \ -F "file=@image.jpg" \ -F "target_format=png"

Downloader API

Extract video and audio from 1000+ platforms like YouTube, TikTok, and Instagram. Includes rich metadata retrieval and multiple quality options.

POST /download
curl -X POST https://api.hqmx.net/downloader/download \ -H "Content-Type: application/json" \ -d '{"url": "https://youtube.com/watch?v=..."}'
